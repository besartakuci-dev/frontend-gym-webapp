# Gym Management Frontend

This is a simple `Vue 3 + PrimeVue` Members CRUD example for a Gym Management System.

## What is included

- Members table
- Add member dialog
- Edit member dialog
- Delete member button
- Local sample data inside the page component

## Run the project

1. `npm install`
2. `npm run dev`

## Main file to study

- `src/views/MembersView.vue`

This file contains the main CRUD logic in a beginner-friendly way.

## Backend

The ASP.NET Core backend is in:

- `backend/GymManagement.Api`

It contains a beginner-friendly Members CRUD API using in-memory data.
